## Installation

- Clone the repository:

    ```bash
    git clone https://github.com/PhungVietBac/AITripSystem.git
    cd ai_trip_system
    ```
    
- Install the required dependencies:

    ```bash
    pip install -r requirements.txt
    ```
## Inference

### Web application
- Run the web server:
```bash
npm run start
```
- Open your web browser and go to `localhost:4000`.
